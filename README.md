# grib2_to_csv

## Requirement
Python 3.8

## Quick Start

```
$ pip3 install git+https://github.com/DEE-BRI/grib2-to-csv.git@main
$ grib2_to_csv --help
```

## GRIB2

See NCEP WMO GRIB2 Documentation (https://www.nco.ncep.noaa.gov/pmb/docs/grib2/grib2_doc/)

## GDAL

Use GDAL (https://gdal.org/)

## Code Sytling

$ pycodestyle src
